# 101511990_comp3123_labtest2 – Weather App

This is my Lab Test 2 project for **COMP3123**.
The app is a simple weather viewer built with **ReactJS** and the **OpenWeatherMap API**.
It allows users to search any city and see real-time weather details.

---

## Features

- Built using **ReactJS (Create React App)**
- Search bar to look up any city
- Displays:

  - Temperature
  - Feels like
  - Humidity
  - Pressure
  - Wind speed
  - City + Country
  - Weather description + icon

- Custom UI using gradient background, Poppins font and weather icons
- API integration using **OpenWeatherMap Current Weather API**

---

## How to Run

1. Download or clone this repository.
2. Open the project folder (the one containing `package.json`).
3. Install dependencies:

```
npm install
```

4. Create a `.env` file in the **root folder** and add your API key or use mine as mentioned in env file or below:

```
REACT_APP_OPENWEATHER_API_KEY=9696700bec48cd72add2eb97eb894898
```

5. Start the application:

```
npm start
```

The app will run at:

```
http://localhost:3000
```
